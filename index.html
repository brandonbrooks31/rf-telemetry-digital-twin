How Data Flows — Every Aircraft Type
Regardless of entry point, all data funnels through the same real-time scoring pipeline.
Step 1

Signal Capture

SDR edge node, ARINC bridge, or direct ACARS feed extracts RSSI, SNR, BER, Doppler, TxPower, and Carrier Deviation.

6 sensor dimensions / packet
Step 2

Secure Transmission

Data POST'd to Google Pub/Sub over OIDC-secured HTTPS. Token verification prevents replay attacks even on cellular links.

POST /pubsub · OIDC verified
Step 3

Mahalanobis Scoring

Cloud Run computes S(x) against an ITU-R calibrated baseline. Score returned in less than 50ms.

chi² threshold: 2.449 / 3.162
Step 4

Alert + Work Order

NOMINAL → Firestore log. WARNING → Slack alert. CRITICAL → ERP webhook fires a draft work order automatically.

ERP webhook · Firestore audit trail
Step 5

Mechanic Takes Action

Pre-flight inspection team sees a work order 6–72 hours before the flight — while the aircraft is still at gate.

Avg lead time: 48 hours
